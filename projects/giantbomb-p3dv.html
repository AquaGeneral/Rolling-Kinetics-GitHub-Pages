Giant Bomb: Polygonally 3D Viewer

Search and view virtually any image accessible from Giant Bomb in an interactive and 3D view.

I wanted to create something with the Giant Bomb API, and my first general idea was to make a photo viewer—almost like a screensaver. One of the most enjoyable parts of the project was balancing the retrieval of resulting images from a search input. I had to make sure I wasn't blocking any code to ensure the user doesn't grow sick of waiting for the images to be downloaded.
